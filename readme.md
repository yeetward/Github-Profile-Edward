### 🔹 AI-Powered Content Analysis Backend (Group Internship Project)

**Python, Django, NLP, MongoDB** | Dec 2025

* Designed MongoDB aggregation pipelines for scalable data processing
* Exposed RESTful APIs for frontend analytics dashboardsTHE ROLE

If you're looking for a large team, a clear career ladder, and well-defined boundaries to your role: this isn't it.

If you want to ship real features to real users from week one, work alongside AI tools that make you faster and sharper, and grow with a company that's going somewhere: read on.

RiPLTec is a Sydney-based SaaS company building the platform infrastructure that professional bodies use to recognise and credential expertise. Our clients include leading professional associations in Australia and the UK, with Europe and Asia on the horizon. We're small, we move fast, and every line of code we ship reaches real users.

WHAT MAKES US DIFFERENT

We've built our development workflow around AI-assisted engineering, specifically Claude Code, combined with a disciplined spec-to-production pipeline. You don't need prior experience with this way of working, but you need to be genuinely excited about it.

If you think AI tools are a threat to developers, this probably isn't your place. If you think they're a force multiplier for people who know what they're doing, keep reading.

THE STACK

.NET 10 / C# 14 / ASP.NET Core · Next.js (App Router) · React + TypeScript · Tailwind CSS · PostgreSQL · Azure Container Apps · GitHub Actions

WHAT YOU'LL ACTUALLY DO

Work across the full stack from day one

Contribute to features end-to-end, from understanding the spec through to deployment

Review and critically evaluate AI-generated code for correctness, security, and architecture fit

Write tests, commit clean code, and push to production within your first weeks, not months

Contribute to documentation, functional specs, and architecture decisions

Cover for colleagues and pick up tasks outside your immediate lane when the team needs it

WHAT WE NEED FROM YOU

Required:

Solid C# fundamentals: generics, async/await, LINQ, pattern matching

React + TypeScript: hooks, component composition, typed interfaces

SQL you can actually write: joins, CTEs, subqueries

Git discipline: branching, pull requests, code reviews

Clear written communication; you'll work from specs, not hand-holding

A critical eye for AI-generated output: you'll use it daily, and you'll need to know when it's wrong

Minimum 1-3 years of professional development experience

Current Australian work rights are essential and based in Sydney

Desirable (not blockers):

Next.js App Router, TanStack Query, Entity Framework Core

Docker, GitHub Actions, Azure basics

Clean Architecture or Domain-Driven Design awareness

WCAG 2.1 AA accessibility knowledge
